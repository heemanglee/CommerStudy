## 학습 방법

CommerStudy는 **강의 섹션별 학습 → 코드 구현 → 위키 정리**의 사이클로 진행됩니다.

| 구분 | 설명 |
|------|------|
| **섹션 학습** | 주차별 강의 시청 및 필기 |
| **코드 구현** | 실습 및 예제 코드 작성 |
| **정리** | Wiki에 개념, 코드 흐름, 회고 기록 |
